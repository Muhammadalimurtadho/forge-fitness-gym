# Forge Fitness -- Gym Landing Page

Industrial-style website for a gym & personal training studio. Designed with raw, bold aesthetics and high-energy visuals -- no generic AI templates.

## Features
- Class schedule table (easy to read, industrial grid)
- Membership pricing comparison cards
- Photo gallery (masonry layout)
- Before-after transformation slider
- "Book Free Trial" CTA in multiple sections
- Fully responsive (mobile-first)
- Font Awesome icons (no emoji)

## Tech Stack
- HTML5
- CSS3 (Grid, Flexbox, custom properties)
- Vanilla JavaScript (slider, smooth scroll)

## Design Philosophy
- **Vibe:** Raw-energik, gritty industrial, "forge/workshop" aesthetic
- **Colors:** Black (#0a0a0a) + Burning Orange (#FF4F00)
- **Fonts:** Bebas Neue + Oswald (zero generic Inter/Roboto/Poppins)
- **No AI-generic patterns:** No centered hero blob, no purple gradients, no templated section order
